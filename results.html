Von Bertalanffy Growth Model Analysis

Author

Gustav Sundvor

Introduction

Analysis of herring growth using the Von Bertalanffy Growth Model (VBGM).

Methods

Data were fitted using the Von Bertalanffy Growth Model equation: \[L_t = L_∞(1 - e^{-K(t-t_0)})\]
